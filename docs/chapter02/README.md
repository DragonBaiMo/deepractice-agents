# 第二章 智能体发展史

理解现代智能体的形态与设计思想，需要回溯历史：从符号主义的逻辑规则体系，到分布式协作智能的转折，再到学习范式如何彻底改变智能体获取能力的方式。

**核心洞察：每一个新范式的出现，都是为了解决上一代范式的核心"痛点"。**

## 2.1 符号主义时代

### 2.1.1 物理符号系统假说 (PSSH)

**纽厄尔和西蒙（1976）的核心论断：**
- **充分性**：物理符号系统具备产生通用智能的充分手段
- **必要性**：任何通用智能系统本质上都是物理符号系统

**核心主张**：智能的本质 = 符号的计算与处理

### 2.1.2 专家系统

典型架构：知识库 + 推理机 + 用户界面

**推理方式：**
- **正向链**：从已知事实出发，数据驱动
- **反向链**：从假设目标出发，目标驱动

**代表案例：MYCIN** - 医学诊断专家系统，600条规则，引入置信因子处理不确定性

### 2.1.3 SHRDLU

威诺格拉德（1968-1970）的"积木世界"系统，首次集成：
- 自然语言理解
- 规划与行动
- 记忆与问答

**历史意义**：建立"感知-思考-行动"闭环的早期范式

### 2.1.4 符号主义的根本挑战

| 挑战 | 描述 |
|-----|-----|
| 知识获取瓶颈 | 专家知识难以完整编码 |
| 常识问题 | 无法穷举隐式背景知识 |
| 框架问题 | 难以判断行动后什么不变 |
| 系统脆弱性 | 规则外情况完全失效 |

## 2.2 心智社会理论

**马文·明斯基的革命性观点：**

> "What magical trick makes us intelligent? The trick is that there is no trick. The power of intelligence stems from our vast diversity, not from any single, perfect principle."

### 核心思想

- **智能体 (Agent)**：极简的专门化心智过程（如 LINE-FINDER、GRASP）
- **机构 (Agency)**：多个智能体协作形成的功能单元
- **涌现 (Emergence)**：复杂行为从简单交互中自发产生

### 对多智能体系统的启发

1. **去中心化控制**：无需中央控制器
2. **涌现式计算**：复杂解从简单规则中产生
3. **智能体社会性**：交互协议、协商策略、信任模型

## 2.3 学习范式的演进

### 2.3.1 联结主义

| 维度 | 符号主义 | 联结主义 |
|-----|---------|---------|
| 方向 | 自上而下 | 自下而上 |
| 知识存储 | 显式规则 | 分布式权重 |
| 能力来源 | 预编码 | 学习获得 |

### 2.3.2 强化学习

**核心要素：**
- Agent（智能体）→ Environment（环境）
- State（状态）→ Action（行动）→ Reward（奖励）

**学习目标**：最大化累积奖励（长期回报）

**典型案例**：AlphaGo 通过自我对弈学习最优策略

### 2.3.3 预训练范式

**两阶段学习：**
1. **预训练**：海量数据上自监督学习，获取通用知识
2. **微调**：少量标注数据适配特定任务

**LLM涌现能力：**
- 上下文学习 (In-context Learning)
- 思维链推理 (Chain-of-Thought)

## 2.4 LLM驱动的现代智能体

### 核心组件

```
感知模块 → 规划模块 → LLM推理 → 执行模块
    ↑                              ↓
    ←────── 记忆模块 ←── 工具箱 ←──┘
```

### 工作循环

1. **感知**：接收环境观察
2. **思考**：规划分解 + LLM推理决策
3. **行动**：工具调用执行
4. **观察**：结果反馈 → 新一轮循环

## 2.5 三大思潮融合

| 思潮 | 核心贡献 | 代表成果 |
|-----|---------|---------|
| 符号主义 | 逻辑推理框架 | 专家系统、深蓝 |
| 联结主义 | 感知与学习能力 | CNN、Transformer |
| 行为主义 | 决策与交互学习 | AlphaGo、RL |

**现代智能体 = LLM（联结主义）+ 工具调用（符号操作）+ 环境交互（行为主义）**

## 2.6 小结

- 符号主义：知识+推理，但遇到知识瓶颈和脆弱性
- 心智社会：分布式智能的哲学启发
- 学习范式：从联结主义到强化学习到预训练
- 现代智能体：三大思潮的技术融合
