# 第一章 初识智能体

欢迎来到智能体的世界！**智能体（Agent）** 已成为驱动 AI 技术变革与应用创新的核心概念。

## 1.1 什么是智能体？

智能体是能够通过**传感器**感知**环境**，并**自主**地通过**执行器**采取**行动**以达成目标的实体。

**四个基本要素：**
- **环境**：智能体所处的外部世界
- **传感器**：感知环境状态的能力
- **执行器**：对环境施加影响的能力
- **自主性**：独立决策的能力

### 1.1.1 传统智能体演进

```
简单反射 → 基于模型 → 基于目标 → 基于效用 → 学习型
```

### 1.1.2 LLM 驱动的新范式

| 维度 | 传统智能体 | LLM 智能体 |
|------|-----------|-----------|
| 核心引擎 | 显式规则 | 神经网络 |
| 知识来源 | 人工编码 | 数据学习 |
| 交互方式 | 结构化 | 自然语言 |

LLM 智能体三大能力：**规划推理**、**工具使用**、**动态修正**

### 1.1.3 分类维度

1. **决策架构**：反射→模型→目标→效用→学习
2. **时间反应性**：反应式 / 规划式 / 混合式
3. **知识表示**：符号主义 / 亚符号主义 / 神经符号主义

## 1.2 运行原理

### 1.2.1 PEAS 模型

- **P**erformance（性能度量）
- **E**nvironment（环境）
- **A**ctuators（执行器）
- **S**ensors（传感器）

### 1.2.2 Agent Loop

感知 → 思考（规划+工具选择）→ 行动 → 观察 → 循环

### 1.2.3 TAO 范式

```
Thought: [思考过程]
Action: [工具调用]
Observation: [执行结果]
```

## 1.3 协作模式

**开发者工具**：Copilot、Claude Code、Cursor

**自主协作者**：单Agent循环、多Agent协作、控制流架构

**Workflow vs Agent**：预定义流程 vs 动态推理决策

## 1.4 小结

- 智能体 = 感知 + 推理 + 行动
- 从传统到 LLM 驱动的演进
- Agent Loop 是核心运行机制
